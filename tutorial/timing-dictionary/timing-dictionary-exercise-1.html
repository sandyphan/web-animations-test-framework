Exercise 1 - Start in the middle

Now you know how to create basic animations. We can now try to add some more timing properties into the animation

Let's now make animation move 300px from left and 400px from top (i.e. moving diagonally), and it will move for 5 seconds and will start after a 2 seconds into the animation.

Hint: use startTime

Show Solution
new Animation(document.querySelector("#a"), {left: "300px", top: "400px"}, {startTime: 2, duration: 5});
