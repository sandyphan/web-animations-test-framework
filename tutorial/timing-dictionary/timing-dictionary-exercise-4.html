Exercise 4 - Repetition

There are times when we want to repeat the animation for a number of times, this is when 'iterationCount' comes in.

For this exercise, let the animation move 300px from left for 3 times.

Show Solution
new Animation(document.querySelector("#a"), {left: "300px", top: "400px"}, {iterationCount: 3, duration: 5});
