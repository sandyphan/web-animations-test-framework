Exercise 5 - Changing Colours

Now you have known a tonne of basics stuffs about web aniamtions, how about let's try out changing the colour of the block since red doesn't look as nice. The question is, how?

It is easy, we only need to access it's background-color property and change it. Sounds easy enough? The following code will change the block from red to green:

new Animation(document.querySelector("#a"), {"background-color": ["red", "green"]}, 2);

So the above code had just access the background-color property in CSS and change it from red to green. You can always change them to other colours such as blue, grey, yellow, or even #827031.

Now let's give it a try. Your task for this exercise is to change the colour of the block from red to green, then to light steel blue, then green then back to red again in 10 seconds.

Hint: you will only need one line of code so don't write 3 lines of codes for changing 3 colours. You can actually put 2+ colours in the square brackets.


Show Solution
new Animation(document.querySelector("#a"), {"background-color": ["red", "green", "lightsteelblue", "green", "red"]}, 10);
