Web Animations Tutorial

Web animations is to have a flash or GIF animation intergrated onto the web page. Normal web animations still require the use of plugins such as Java Applets, Shockwave, Adobe Flash. In this tutorial, we will be showing you how to create animations using javascript without the need of installing plugins.


Timing Functions

Timing functions allow time scaling which adds effects such as easing and stepping to animations. To create a basic animation with timing functions, please follow the following interface:

new Animation(element, {effects}, {timingFunction: new TimingFunction("ease"), duration: #});
new Animation(element, {effects}, {timingFunction: new StepingTimingFunction(1, 'start'), duration: #});

Easing effects allow animations to have slow start and endings which makes the animations appear to be more smooth. It was defaulted to linear. There are 5 easing effects:

  • linear: animation runs at normal speed with no timing scaling.
  • ease: slow start for a quarter of the animation time and reduce speed of animation at quarter of the animation time.
  • ease-in: slow start for 2 fifths of the animation time then runs at normal speed.
  • ease-out: run at normal speed then ends slowly for the last 2 fifths of total duration of the animation.
  • ease-in-out: slow start for 2 fifths of the animation time then runs at normal speed for a short time, then ends slowly for the last 2 fifths of the total duration of the animation.

Stepping functions scale the timed item's timing so that the timed item occurs in a series of discete steps. There are 3 stepping effects:

  • start: the change in value occurs at the beginning of the interval.
  • middle: the change in value occurs at the midpoint of the interval.
  • end: the change in value occurs at the end of the interval.
