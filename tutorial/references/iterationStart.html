iterationStart

iterationStart is of type double.

A finite real number greater than or equal to zero representing the number of iterations into the timed item at which to begin.

For instance, a value of 0.5 would cause the timed item to begin half-way through the first iteration.

iterationCount is added to the iterationStart. That is, if iterationCount is 2 and iterationStart is 0.5, then the item will repeat twice with begin and end half-way through the timed item's interation interval.

It has the following exeptions:

  • DOMException of type IndexSizeError
    • Raised on setting if the value to set is a real number less than zero.
