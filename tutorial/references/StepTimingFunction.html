StepTimingFunction()

StepTimingFunction() is used to scale a timed item's timing such that the timed item occurs in a series of discrete steps. It is included as part of the timing dictionary.

StepTimingFunction() has the following interface with the following properties:

StepTimingFunction(numSteps, position)
Parameter Type Nullable Optional Description
numSteps unsigned integer No No A number greater than or equal to one representing the number of discrete steps in the function.
Exception:
DOMException of type IndexSizeError (Raised on setting if the number of steps is zero);
position StepPosition No Yes (Default to 'end') The point within each interval at which the change in value occurs.
start: the change in value occurs at the beginning of the interval.
middle: the change in value occurs at the midpoint of the interval.
end: the change in value occurs at the end of the interval.
