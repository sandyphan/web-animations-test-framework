SeqGroup()

SeqGroup() is used in grouping animations into a sepuential group such that the children inside the group share common timing properties/timing functions and they run in order.

SeqGroup() has the following interface with the following properties:

new SeqGroup([children, timing]);

Sequence group has properties very similar to parallel groups. The following are the properties of sequence group:

  1. A new sequence group object is created
  2. It will establish a temporary timing dictionary based on the following:
    • If the input timing is a TimingDictionary object:
      • Then timing dictionary refers to timing.
    • If the input timing is a double:
      • Then timing dictionary will be a new timing dictionary object with all properties set to default values and iterationDuration set to timing.
    • If the input timing is null:
      • Then timing dictionary will be a new timing dictionary object with all properties set to default values.
  3. 'children' is nullable and compulsory.
  4. 'timing' is nullable and optional.
