iterationCount

iterationCount is of type unrestricted double.

A real number greater than or equal to zero (including positive infinity) representing the number of times to repeat the timed item.

For instance, a value of 2 would repeat 2 times and a value of 2.5 would repeat for 2 times and a half, hence the timed item will repeat 2 times and ends half-way through it's duration.

It has the following exeptions:

  • DOMException of type IndexSizeError
    • Raised on setting if the value to set is a real number less than zero.
