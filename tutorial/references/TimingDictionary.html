TimingDictionary

TimingDictionary is a dictionary of timing objects.

TimingDictionary dictionary is not a compulsory component in animations and groups definition. All timing items should be placed inside {} to specify that the input timing is a dictionary. TimingDictionary dictionary has the following acceptable members:

  • startTime
  • startDelay
  • fillMode
  • iterationStart
  • iterationCount
  • iterationDuration
  • defaultPlaybackRate
  • direction
  • timingFunction
