new Animation()

Animation() is an object that is a type of timed item that apply an animation effect to a target element.

It has the following interface:

new Animation(element, {effect}, {timing});

With the followng properties:

Parameter Type Nullable Optional Description
element Animation Target Yes No The target element in which animations will apply onto.
effect object Yes No

The effects to be apply onto the newly created Animation object.

If this parameter is null, the newly created Animation object will have no effects apply onto it.
timing double or TimingDictionary Yes Yes The timing properties of the sequence group
  • If parameter is a double:
    • create a new timing dictionary object with all properties set to their default values.
    • iterationDuration is set to the input double value.

  • If parameter is a TimingDictionary:
    • Then timing dictionary is set to the input timing.

  • If parameter is a nul:
    • create a new timing dictionary object with all properties set to their default values.
