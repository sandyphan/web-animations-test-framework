new Animation()

Animation() is an object that is a type of timed item that apply an animation effect to a target element.

It has the following interface:

new Animation(element, {effect}, {timing});

With the followng properties:

  • new Animation:
    • Creates a new animation object: animation
    • Establish a temporary timing dictionary based on the following:
      • If input timing is a TimingDictionary object:
        • Let timing dictionary refer to timing.
      • If input timing is a double:
        • Create a new timing dictionary object with all properties set to default values and iterationDuration set to timing.
      • If input timing is null/undefined object:
        • Create a new timing dictionary object with all properties set to default values.
    • Assign animation effects as follow:
      • If effect is an AnimationEffect object or a CustomAnimationEffect object:
        • Assign animation.effect to effect.
      • If effect is null:
        • Set animation.effect to null
