Sequential Animation Group
Exercise 3 - More Challenges

Now you know how to use both parallel and sequential animation groups. Let's get started on some harder exercises.

For this exercise, create 4 animations to 4 different animation divs. Divide the animations evenly into 2 groups where the first group contains the 1st and 2nd elements. The second group should wait until the first group has finished running to run. The elements of the first group should move 300px and 400px from left respectively. The group should run for 3 seconds and repeat twice. The second group should move 300px and 400px from left respectively where the 2nd element in the group will have to wait half way to start. That is each element should run for 4 seconds and the 2nd element will wait for 2 seconds to run. The second group should also repeat itself 3 times. In addition, the first group will change colours from red to green then back to red, the second group will change colours from blue to yellow then to blue.

  • Create 4 animations.
  • Create 2 groups, group the 1st and 2nd elements into the first and the rest into the second.
  • Animate the first group such that it repeats twice. Elements in this group should run in parallel.
  • Animate the second group such that one runs half way to anther and repeats itself 3 times. Elements should run in sequence
  • First group changes colour from red to green then to red.
  • Second group changes colour from blue to yellow then to blue.
Show Solution
var A = new Animation(document.querySelector("#a"), {left: "300px", "background-color": ["red", "green", "red"]}, 3); var B = new Animation(document.querySelector("#b"), {left: "400px", "background-color": ["red", "green", "red"]}, 3); var group1 = new ParGroup([A, B], {iterationCount: 2}); var C = new Animation(document.querySelector("#c"), {left: "300px", "background-color": ["blue", "yellow", "blue"]}, 4); var D = new Animation(document.querySelector("#d"), {left: "400px", "background-color": ["blue", "yellow", "blue"]}, {startDelay: -2, duration: 4}); var group2 = new SeqGroup([C, D], {iterationCount: 3}); new SeqGroup([group1, group2]);
