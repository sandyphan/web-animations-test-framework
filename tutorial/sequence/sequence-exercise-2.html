Sequential Animation Group
Exercise 2 - Overlapping (Challenge)

Let's have some play around with start delay in sequence groups.

In this exercise, create 3 different animations using 3 different animation divs, make the first and second animation overlap by 2 seconds and make the third animation waits 2 seconds after the first and second animation has finished running.

  • The first animation moves 400px from left
  • Start position of the second animation div should be at 200px from left
  • Let the second animation move to 500px from left
  • Let the third animation start at 100px from top
  • The third animation should move to 300px from left
  • All 3 animations should run for 5 seconds
  • The whole cycle should be repeated twice.
Show Solution
var A = new Animation(document.querySelector("#a"), {left: "400px"}, 5);
var B = new Animation(document.querySelector("#b"), {left: ["200px", "500px"]}, {startDelay: -2, duration: 5});
var C = new Animation(document.querySelector("#c"), {left: "300px"}, {startDelay: 2, duration: 5});
var seq = new SeqGroup([A, B, C], {iterationCount: 2});
seq.play();
